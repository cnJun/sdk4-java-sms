SDK4 SMS Libraries
==================

# 1. Overview

sdk4.com 短信服务，集成了常用的第三方短信通道，比如：阿里云短信、云片等。

# 2. Usage

Maven : 
```
<dependency>
    <groupId>com.sdk4</groupId>
	<artifactId>sdk4-java-sms</artifactId>
	<version>0.0.1</version>
</dependency>
```
